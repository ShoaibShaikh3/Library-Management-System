# Library Management System

A Python-based Library Management System developed as a learning project to understand Object-Oriented Programming, file handling, and database concepts.

## Features

* Add new books
* View available books
* Search books
* Issue books
* Return books
* Delete books
* Store book records

## Technologies Used

* Python
* MySQL (optional)
* OOP Concepts

## Future Improvements

Planned features for future versions:

* Tkinter GUI Interface
* User Authentication System
* HTML/CSS Frontend
* Flask/Django Web Version
* Advanced Search & Filtering
* Book Borrowing History
* Dashboard & Analytics

## Learning Objectives

This project was built to improve:

* Python Programming
* Database Management
* CRUD Operations
* Software Development Practices
